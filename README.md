# Adham Erlangga Siwi Portfolio

Static portfolio website for GitHub Pages.

## Local preview

Open `index.html` directly in a browser, or run a small static server from this folder:

```powershell
python -m http.server 4173
```

Then open `http://localhost:4173`.

## Deployment recommendation

Use GitHub Pages first. This site is plain HTML, CSS, and JavaScript, so it does not need Vercel, Node.js, npm, or a build step.

For a GitHub user site, push these files to a repository named:

```text
adhamerlangga.github.io
```

Then set Pages to publish from the `main` branch root.

## Editing content

- Main text, projects, links: `index.html`
- Visual design and responsive layout: `styles.css`
- Navigation, project carousel, canvas animation: `script.js`
